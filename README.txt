Данное задание:
Написать программу, принимающую три значения с консоли (a, b и c) и выполняющую следующие вычисления с функцией a * sin(b * value) + (2 / c):
	1. Запрашивает значение value у пользователя и выводит значение функции;
	2. Запрашивает два значения firstValue и secondValue и находит наибольшее значение функции; выводит его;
	3. Запрашивает два значения firstValue и secondValue, а также значение шага stepValue. Находит значение функции от firstValue, всех функций от firstValue до firstValue + stepValue * counter (пока (firstValue + stepValue * counter) < secondValue) и значение функции от secondValue; выводит их все.
